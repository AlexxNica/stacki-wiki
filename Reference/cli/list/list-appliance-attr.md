## list appliance attr

### Usage

`stack list appliance attr [appliance]`

### Description

Lists the set of attributes for appliances.

### Arguments

* `{appliance}`

   Name of appliance


### Examples

* `stack list appliance attr compute`

   List the attributes for compute appliances
