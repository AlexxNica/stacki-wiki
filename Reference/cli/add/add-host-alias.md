## add host alias

### Usage

`stack add host alias {host} {alias=string}`

### Description

Adds an alias to a host

### Arguments

* `[host]`

   Host name of machine


### Parameters
* `[alias=string]`

   Alias for the host.

### Examples

* `stack add host alias backend-0-0 alias=b00`

   Add a host alias "b00" to host "backend-0-0".
