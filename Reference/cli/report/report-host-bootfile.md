## report host bootfile

### Usage

`stack report host bootfile {host ...} {action=string}`

### Description

Output the PXE file for a host

### Arguments

* `[host]`

   One or more hostnames


### Parameters
* `[action=string]`

   Generate PXE file for a specified action
