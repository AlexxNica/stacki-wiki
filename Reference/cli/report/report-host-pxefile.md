## report host pxefile

### Usage

`stack report host pxefile {host ...} {action=string}`

### Description

Output the PXE file for a host

### Arguments

* `[host]`

   One or more hostnames


### Parameters
* `[action=string]`

   Generate PXE file for a specified action
