## set host appliance

### Usage

`stack set host appliance {host ...} {appliance=string}`

### Description

Set the Appliance for a list of hosts.

### Arguments

* `[host]`

   One or more host names.


### Parameters
* `[appliance=string]`

   Appliance name (e.g. "backend").
