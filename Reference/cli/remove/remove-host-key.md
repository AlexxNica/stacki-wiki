## remove host key

### Usage

`stack remove host key {host} [id=string]`

### Description

Remove a public key for a host.

### Arguments

* `[host]`

   A host name.


### Parameters
* `{id=string}`

   The ID of the key you wish to remove. To get the key id, execute:
	"rocks list host key"
