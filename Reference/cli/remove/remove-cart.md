## remove cart

### Usage

`stack remove cart {cart ...}`

### Description

Remove a cart from both the database and filesystem.

### Arguments

* `[cart]`

   List of carts.


### Examples

* `stack remove cart devel`

   Remove the cart named 'devel'.
