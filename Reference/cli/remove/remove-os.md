## remove os

### Usage

`stack remove os {os}`

### Description

Remove an OS definition from the system.

### Arguments

* `[os]`

   The OS type (e.g., "linux", "sunos").


### Examples

* `stack remove os sunos`

   Removes the OS type "sunos" from the database.
