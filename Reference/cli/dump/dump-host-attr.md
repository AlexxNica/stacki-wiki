## dump host attr

### Usage

`stack dump host attr [host]`

### Description

Dump the set of attributes for hosts.

### Arguments

* `{host}`

   Host name of machine


### Examples

* `stack dump host attr compute-0-0`

   Dump the attributes for compute-0-0.
