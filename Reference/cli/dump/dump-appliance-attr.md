## dump appliance attr

### Usage

`stack dump appliance attr [appliance]`

### Description

Dump the set of attributes for appliances.

### Arguments

* `{appliance}`

   Name of appliance


### Examples

* `stack dump appliance attr compute`

   List the attributes for compute appliances
