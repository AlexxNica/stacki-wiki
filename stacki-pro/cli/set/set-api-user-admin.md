## set api user admin

### Usage

`stack set api user admin {Username} [admin=string]`

### Description

Set or unset admin privileges of a user.

### Arguments

* `[Username]`

   Username of user for which to set / unset
	the admin privileges.


### Parameters
* `{admin=string}`

   Set or unset admin privileges.
