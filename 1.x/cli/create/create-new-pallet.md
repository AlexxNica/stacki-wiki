## create new pallet

### Usage

`stack create new pallet {version} {name} {color} [color=string] [name=string] [version=string]`

### Description

Create a new pallet from a template.

### Arguments

* `{version}`

   

* `{name}`

   

* `{color}`

   


### Parameters
* `[color=string]`

   
* `[name=string]`

   
* `[version=string]`

   

### Examples

* `stack create new pallet`
