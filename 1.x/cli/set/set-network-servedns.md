## set network servedns

### Usage

`stack set network servedns {network} {servedns} [servedns=bool]`

### Description

Sets/Unsets the capability for serving DNS
	for a given subnet

### Arguments

* `{network}`

   Name of the Network

* `{servedns}`

   True/False


### Parameters
* `[servedns=bool]`

   True/False
