## set network zone

### Usage

`stack set network zone {network} {zone} [zone=string]`

### Description

Set the zone/domain name associated with a subnet

### Arguments

* `{network}`

   Network Name

* `{zone}`

   Zone / Domain that the network belongs to.
	Example: optiputer.net


### Parameters
* `[zone=string]`

   Zone / Domain that the network belongs to.
	Example: optiputer.net
