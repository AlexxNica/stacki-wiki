.. _remove.access:

remove access
-------------

Usage
"""""

``rocks remove access [command=None] [group=None]``

Description
"""""""""""

        Remove Access control pattern.
	

Parameters
""""""""""
``[command=None]``

	Command Pattern.
	
``[group=None]``

	Group name / ID for access.
	

Examples
""""""""

``rocks remove access '*' apache``


	Remove "apache" group access to all "rocks" commands
	

``rocks remove access command="list*" group="wheel"``


	Remove "wheel" group access to all "rocks list" commands
