.. _report.host.storage.controller:

report host storage controller
------------------------------

Usage
"""""

``rocks report host storage controller {host}``

Description
"""""""""""

	Output the storage controller configuration for a specific host

	

Arguments
"""""""""
``{host}``

	One host name.
	


Examples
""""""""

``rocks report host storage controller compute-0-0``


	Output the storage controller configuration for compute-0-0.
