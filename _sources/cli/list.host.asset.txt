.. _list.host.asset:

list host asset
---------------

Usage
"""""

``rocks list host asset [host]... [dmitype]``

Description
"""""""""""

	Calls 'dmidecode' and outputs the info from the supplied 'DMI type'.

	

Arguments
"""""""""
``[host]``

	Zero, one or more host names. If no host names are supplied, info about
	all the known hosts is listed.
	

``[dmitype]``

	The 'DMI type' to output (see the dmidecode man page for a list of
	types).
	Default: 1. '1' is the 'System Information'.
	


Examples
""""""""

``rocks list host asset compute-0-0``


	List the DMI System Information for compute-0-0.
	

``rocks list host asset compute-0-0 dmitype=3``


	List the DMI information associated with DMI type 3 ('Chassis
	Information') for compute-0-0.
