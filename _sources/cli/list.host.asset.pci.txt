.. _list.host.asset.pci:

list host asset pci
-------------------

Usage
"""""

``rocks list host asset pci [host]... [search=string]``

Description
"""""""""""

	List asset information about a host's PCI interfaces.

	

Arguments
"""""""""
``[host]``

	Zero, one or more host names. If no host names are supplied, info about
	all the known hosts is listed.
	


Parameters
""""""""""
``[search=string]``

	Output host PCI info that matches the 'search'.
	

Examples
""""""""

``rocks list host asset pci compute-0-0``


	List PCI info for compute-0-0.
	

``rocks list host asset pci search="manufacturer=Broadcom Corporation"``


	List PCI info for all hosts that have PCI cards manufactured by "Broadcom Corporation".
