.. _add.distribution:

add distribution
----------------

Usage
"""""

``rocks add distribution {distribution} [graph=string] [os=string]``

Description
"""""""""""

	Add a distribution specification to the database.

	

Arguments
"""""""""
``{distribution}``

	Name of the new distribution.
	


Parameters
""""""""""
``[graph=string]``

	Graph associated with the distribution
	
``[os=string]``

	OS associated with the distribution
	

Examples
""""""""

``rocks add distribution rocks-dist``


	Adds the distribution named "rocks-dist" into the database.
