.. _list.appliance:

list appliance
--------------

Usage
"""""

``rocks list appliance [appliance]...``

Description
"""""""""""

	Lists the appliances defined in the cluster database.
	
	

Arguments
"""""""""
``[appliance]``

	Optional list of appliance names.
	


Examples
""""""""

``rocks list appliance``


	List all known appliances.
