list appliance attr
-------------------

Usage
"""""

``rocks list appliance attr [appliance]``

Description
"""""""""""

	Lists the set of attributes for appliances.

	

Arguments
"""""""""
``[appliance]``

	Name of appliance
	


Examples
""""""""

``rocks list appliance attr compute``
