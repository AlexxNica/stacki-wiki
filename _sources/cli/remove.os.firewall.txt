.. _remove.os.firewall:

remove os firewall
------------------

Usage
"""""

``rocks remove os firewall {os} [rulename=string]``

Description
"""""""""""

	Remove a firewall rule for an OS type. To remove
	a rule, one must supply the name of the rule.

	

Arguments
"""""""""
``{os}``

	Name of an OS type (e.g., "linux", "sunos").
	


Parameters
""""""""""
``[rulename=string]``

	Name of the OS-specific rule
