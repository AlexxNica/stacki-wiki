.. _run.roll:

run roll
--------

Usage
"""""

``rocks run roll [roll]...``

Description
"""""""""""

	Installs a Roll on the fly
	
	

Arguments
"""""""""
``[roll]``

	List of rolls. This should be the roll base name (e.g., base, hpc,
	kernel).
	


Examples
""""""""

``rocks run roll viz``

		
	Installs the Viz Roll onto the current system.
