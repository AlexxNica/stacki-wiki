dump appliance attr
-------------------

Usage
"""""

``rocks dump appliance attr [appliance]``

Description
"""""""""""

	Dump the set of attributes for appliances.

	

Arguments
"""""""""
``[appliance]``

	Name of appliance
	


Examples
""""""""

``rocks dump appliance attr compute``


	List the attributes for compute appliances
