.. _list.user.widget:

list user widget
----------------

Usage
"""""

``rocks list user widget [user]...``

Description
"""""""""""

	Lists a users access to the management console widgets.

	

Arguments
"""""""""
``[user]``

	Zero, one or more user names (or UIDs). If no names are supplied, 
	info about all the known users are listed.
	


Examples
""""""""

``rocks list user widget root``


	List the widget the root account has access to.  By default this
	is always the complete set of widgets.
	

``rocks list user widget``


	List info for all known users.
