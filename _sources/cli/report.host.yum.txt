.. _report.host.yum:

report host yum
---------------

Usage
"""""

``rocks report host yum {host}``

Description
"""""""""""

	Create a report that describes the yum.conf repo file that should be
	put on hosts.

	

Arguments
"""""""""
``{host}``

	Host name of machine
	


Examples
""""""""

``rocks report host yum compute-0-0``


	Create a report of the yum.conf Rocks repo for compute-0-0.
