.. _add.host.message:

add host message
----------------

Usage
"""""

``rocks add host message [host]... {message} [channel=string]``

Description
"""""""""""

        Adds a message to one or most host Message Queues

	

Arguments
"""""""""
``[host]``

	Zero, one or more host names. If no host names are supplied, the
        message is sent to all hosts.
	

``{message}``

	Message text
	


Parameters
""""""""""
``[channel=string]``

	Name of the channel
	

Examples
""""""""

``rocks add host message compute-0-0 "hello world" channel=debug``


	Sends "hello world" over the debug channel using the Message
        Queue on compute-0-0.
