.. _load.storage.partition:

load storage partition
----------------------

Usage
"""""

``rocks load storage partition [file=string] [processor=string]``

Description
"""""""""""

	Output partitioning commands for a host. These commands could either
	be hardware disk controller commands or Red Hat partitioning commands.
	
	

Parameters
""""""""""
``[file=string]``

	The file that contains the partitioning data.
	
``[processor=string]``

	The processor used to parse the file.
	Default: default.
	

Examples
""""""""

``rocks load partfile file=partitions.csv``


	Read partition info from partitions.csv and use the default processor
	to parse the data.
