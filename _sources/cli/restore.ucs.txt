.. _restore.ucs:

restore ucs
-----------

Usage
"""""

``rocks restore ucs [host]``

Description
"""""""""""

	Import the logical and system XML configuration into UCS Manager

	

Arguments
"""""""""
``[host]``

	Provide one UCS Manager host name.
	


Examples
""""""""

``rocks import ucs ucs-manager-0-0 mybackup.xml``


	Import the mybackup.xml configuration into ucs-manager-0-0 host.
