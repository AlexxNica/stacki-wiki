.. _list.host.alert:

list host alert
---------------

Usage
"""""

``rocks list host alert [host]...``

Description
"""""""""""

        List all the queued up alert messages for one or more
        hosts.

	

Arguments
"""""""""
``[host]``

	Zero, one or more host names. If no host names are supplied, list
        alerts for all hosts.
	


Examples
""""""""

``rocks list host alert compute-0-0``


	List all alerts for compute-0-0
