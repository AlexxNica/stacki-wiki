.. _list.host.asset.network:

list host asset network
-----------------------

Usage
"""""

``rocks list host asset network [host]... [search=string]``

Description
"""""""""""

	List asset information about a host's network interfaces.

	

Arguments
"""""""""
``[host]``

	Zero, one or more host names. If no host names are supplied, info about
	all the known hosts is listed.
	


Parameters
""""""""""
``[search=string]``

	Output host network info that matches the 'search'.
	

Examples
""""""""

``rocks list host asset pci compute-0-0``


	List network info for compute-0-0.
	

``rocks list host asset network search="manufacturer=Broadcom Corporation"``


	List network info for all hosts that have network interfaces manufactured by "Broadcom Corporation".
