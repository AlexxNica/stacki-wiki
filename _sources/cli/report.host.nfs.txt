.. _report.host.nfs:

report host nfs
---------------

Usage
"""""

``rocks report host nfs {host}``

Description
"""""""""""

	Outputs configuration files for NFS.

	

Arguments
"""""""""
``{host}``

	One host name.
	


Examples
""""""""

``rocks report host nfs compute-0-0``


	Output the NFS configuration for compute-0-0.
