.. _add.appliance.route:

add appliance route
-------------------

Usage
"""""

``rocks add appliance route {appliance} {address} {gateway} [netmask=string]``

Description
"""""""""""

	Add a route for an appliance type in the cluster

	

Arguments
"""""""""
``{appliance}``

	The appliance type (e.g., 'compute'). This argument is required.
	

``{address}``

	Host or network address
	

``{gateway}``

	Network or device gateway
	


Parameters
""""""""""
``[netmask=string]``

	Specifies the netmask for a network route.  For a host route
	this is not required and assumed to be 255.255.255.255
