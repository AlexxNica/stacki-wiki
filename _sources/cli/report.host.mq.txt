.. _report.host.mq:

report host mq
--------------

Usage
"""""

``rocks report host mq {host}``

Description
"""""""""""

        Outputs the Rocks Message Queue configuration for a specific host.
        
	

Arguments
"""""""""
``{host}``

        Create MQ configuration for machine named 'host'. If
	no host name is supplied, then generate the configuration file
	for all hosts.
