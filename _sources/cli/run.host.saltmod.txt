run host saltmod
----------------

Usage
"""""

``rocks run host saltmod {command} {args}``

Description
"""""""""""

	Runs a salt module on all compute nodes
	

Arguments
"""""""""
``{command}``

	The salt command that should be run.
	

``{args}``

	The arguments to the salt command.
	


Examples
""""""""

``rocks run saltmod command="rocks-collect.collect" args="cpu.rrd 10sec 1"``


	Runs the rocks-collect.collect command on all hosts. This command fetches
	1 average sample from cpu.rrd database for a 10 sec interval.
