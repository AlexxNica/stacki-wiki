.. _dump.host.attr:

dump host attr
--------------

Usage
"""""

``rocks dump host attr [host]``

Description
"""""""""""

	Dump the set of attributes for hosts.

	

Arguments
"""""""""
``[host]``

	Host name of machine
	


Examples
""""""""

``rocks dump host attr compute-0-0``


	Dump the attributes for compute-0-0.
