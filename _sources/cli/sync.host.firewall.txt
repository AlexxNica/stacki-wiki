.. _sync.host.firewall:

sync host firewall
------------------

Usage
"""""

``rocks sync host firewall [restart=boolean]``

Description
"""""""""""

	Reconfigure and optionally restart firewall for named hosts.

	

Parameters
""""""""""
``[restart=boolean]``

	If "yes", then restart iptables after the configuration files are
	applied on the host.
	The default is: yes.
	

Examples
""""""""

``rocks sync host firewall compute-0-0``


	Reconfigure and restart the firewall on compute-0-0.
