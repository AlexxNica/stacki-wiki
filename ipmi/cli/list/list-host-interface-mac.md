## list host interface mac

### Usage

`stack list host interface mac {host ...}`

### Description

List all NIC's on a machine. Currently works
	for Dell, HP.

### Examples

* `stack list host interface mac compute-0-0`

   List information about all NIC's on compute-0-0.
