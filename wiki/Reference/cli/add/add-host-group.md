## add host group

### Usage

`stack add host group {host ...} {group=string}`

### Description

Adds a group to one or more hosts.

### Arguments

* `[host]`

   One or more host names.


### Parameters
* `[group=string]`

   Group for the host.

### Examples

* `stack add host group backend-0-0 group=test`

   Adds host backend-0-0 to the test group.
