## add api user group

### Usage

`stack add api user group {user} [group=string]`

### Description


	Add users to an existing group
	

### Arguments

* `[user]`

   User


### Parameters
* `{group=string}`

   Group to add the user to.

### Examples

* `stack add user group greg group=staff`

   Adds user 'greg' to the 'staff' group
