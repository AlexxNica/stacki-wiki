## add host route

### Usage

`stack add host route {host ...} [address=string] [gateway=string] [netmask=string] [syncnow=string]`

### Description


	Add a route for a host

	

### Arguments

* `[host]`

   Host name of machine


### Parameters
* `{address=string}`
* `{gateway=string}`
* `{netmask=string}`
* `{syncnow=string}`

   Add route to the routing table immediately
