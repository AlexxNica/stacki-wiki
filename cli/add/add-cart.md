## add cart

### Usage

`stack add cart {cart} [os=string]`

### Description

Add a cart.

### Arguments

* `[cart]`

   The name of the cart to be created.


### Parameters
* `{os=string}`

   The OS you wish to build a cart for (e.g., 'redhat', 'sles', 'ubuntu').
	This will create default OS-specific node and graph XML files in the
	cart.
	Default: redhat.
