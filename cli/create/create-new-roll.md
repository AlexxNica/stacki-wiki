## create new roll

### Usage

`stack create new roll {version} {name} {color} [color=string] [name=string] [version=string]`

### Description

Create a new roll from a template.

### Arguments

* `{version}`

   

* `{name}`

   

* `{color}`

   


### Parameters
* `[color=string]`

   
* `[name=string]`

   
* `[version=string]`

   

### Examples

* `stack create new roll`
