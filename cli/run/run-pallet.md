## run pallet

### Usage

`stack run pallet [pallet]...`

### Description

Installs a pallet on the fly

### Arguments

* `[pallet]`

   List of pallets. This should be the pallet base name (e.g., base, hpc,
	kernel).


### Examples

* `stack run pallet viz`

   Installs the Viz pallet onto the current system.
